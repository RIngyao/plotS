Type of file format supported in plotS

It support comma separated file (csv), tab delimited (tsv) and excel sheet file (xls and xlsx)

Must the input data be of only numeric columns?

No, data can be a combination of numerical and categorical variables.

Why and when to reshape the data?

Reshaping is required depending on the input data so that variables can be compared. In order to represent and compare variables in plotS, the input data must be in long format rather than wide format. In the example below, Table 1 represents the wide format, while Table 2 represent the long format. You need to reshape your data if it is in wide format and want to compare between variables (control, treatment1 and treatment2).

What is aesthetic option?

Aesthetic options mapped the variables to visual properties - color, shape or line type. It let user customize the visual properties and differentiate between variables.

How was effect size estimated?

Effect size is estimated using different methods based on the statistical test applied.

  • T-test: cohen’s d of rstatix package is used for the test.
  • ANOVA : partial eta squared from effectsize package
  • wilcoxon test: the effect size is measured as z statistic, derived from the wilcoxon test, divided by square root of the sample size. It is implemented using rstatix package.
  • Kruskal-Wallis test: measured as eta squared based on H-statistic derived from the Kruskal-Wallis test. It is given as eta2[H] = (H - k + 1)/(n - k). Where, k is the number of groups and n is the total number of observations. It is implemented using rstatix package.
